# Refactored Data Compression Projects

This directory contains refactored, modernized, and English-translated versions of the original Data Compression (KKD) mini-projects.

## Projects

### 1. [Entropy Analyzer](./entropy_analyzer)
**Original:** `1/entropy_analyzer.rs`
**Description:** Calculates Shannon entropy and conditional entropy of files to analyze information content and redundancy.

### 2. [Arithmetic Coder](./arithmetic_coder)
**Original:** `2/arithmetic_coder.cpp`
**Description:** A complete implementation of adaptive arithmetic coding for lossless data compression.

### 3. [Universal Codes LZW](./universal_codes_lzw)
**Original:** `3/main.cpp`
**Description:** LZW compression implementation supporting multiple universal output encodings (Elias Gamma, Delta, Omega, and Fibonacci).

### 4. [Image Predictor Analyzer](./image_predictor_analyzer)
**Original:** `4/main.cpp`
**Description:** Analyzes TGA images using various predictive models (including JPEG-LS) to determine the most effective predictor for entropy reduction.
